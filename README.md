# GET STARTEd

## base code
start server with default values
```cmd
  php D:/php_server/server_handler.php
```

## flags
### `--dir` or `-d`
start server in specific directory
```cmd
  php D:/php_server/server_handler.php --dir <directory>
```

### ``--break`` or `-b`
stop server
```cmd
  php D:/php_server/server_handler.php --break
```
